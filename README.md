# Rent-a-car-backend-app-done-with-REST-API-and-Microservice-architecture
Rent a car app [SQL/REST/Docker/Java/Microservice]

This project has been done by 1 student from University of Novi Sad, Faculty of Mathematics.

Its done as a project for univeristy.

It presents aplication used for rent a car service.

It's done in Java with Hibernate, database in done in SQL.



Features:
  - SQL databse (MySQL)
    - Logic scheme
    - Relation scheme
    - DLL
    - SQL queries
  - Java
    - Hibernate
  - Docker
  - REST API
  - Microservice architecture
