# Rent-a-car-backend-app-done-with-REST-API-and-Microservice-architecture
Rent a car app [SQL/REST/Docker/Java/Microservice]

This project has been done by 1 student from University of Novi Sad, Faculty of Mathematics.

Its done as a project for univeristy.

It presents aplication used for rent a car service.

It's done in Java with Hibernate, database in done in SQL.

All requests are processed with REST using REST controllers.

Project has all validations and Exception handlers needed.

All components(services) in project are indepedent using microservice architecture.

All services are run with Docker.

Inside projet is text file with links to all services on Docker.


Features:
  - SQL databse (MySQL)
    - Logic scheme
    - Relation scheme
    - DLL
    - SQL queries
  - Java
    - Hibernate
  - Docker
  - REST API
  - Microservice architecture
  - Naming service (Eureka)
  - API gateway
